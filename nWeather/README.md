## Getting Started

### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/Kirushan-Balakrishnan/pesto.git
   ```
3. Let’s install Expo using npm
   ```sh
   npm install expo-cli --global
   ```
4. Starting the development server
   ```sh
   expo start
   ```
